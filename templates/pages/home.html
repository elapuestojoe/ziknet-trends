Ziknet Trends

Ziknet Trends is a project developed at ITESM that uses LSTM neural networks to develop a model that predicts Zika Virus epidemics using Google Trends search data.

How does it work?

Information retrieved from search engines reflects a population's particular interest or concern regarding specific topics. Ziknet-Trends is a real time monitoring system that correlates changes in search patterns and previous incidence information to predict when a population is at risk of a Zika outbreak.

The following graphs shows search patterns and the number of Zika cases in Mexico in 2016, there is a subtle yet noticeable correlation between search incidence and the total number of cases.

Why LSTM Networks?

LSTM networks have shown to be very efficient at time-series predictions
